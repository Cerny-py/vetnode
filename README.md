# Vetnode - Franco Ortiz Y Bastian Meneses

# Instrucciones
1. Crear la base de datos copiando el scrip del archivo 'vetnode_ortiz_meneses.sql' en phpmyadmin

2. instalar las dependencias:
    npm install

3. iniciar el servidor:
    npm start
    
4. abrir en el navegador:
    localhost:3000
